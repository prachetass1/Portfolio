## 📸 Features

- ✨ **Animated particle canvas** with constellation-style connecting lines
- 🖱️ **Interactive cursor glow** effect
- ⌨️ **Typewriter animation** cycling through developer roles
- 🔄 **Spinning avatar ring** with floating tech badges
- 📊 **Skill progress bars** that animate on scroll
- 🔢 **Animated stat counters**
- 📱 **Fully responsive** — mobile, tablet & desktop
- 🌙 **Dark theme** with Electric Blue + Violet + Emerald accent palette
- 🧭 **Sticky navbar** with active section highlighting
- 📬 **Contact form** with input validation

## 🚀 Run Locally

```bash
# Clone the repository
git clone https://github.com/prachetass1/Portfolio-Website.git

# Navigate into the folder
cd Portfolio-Website

# Open in browser (any of these)
start index.html                        # Windows
open index.html                         # macOS

# Or serve with Python
python -m http.server 8080
# Then visit http://localhost:8080
```
